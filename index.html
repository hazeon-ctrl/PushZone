## Audio Design System

**Philosophy:** Every action should have a distinct, musically related sound using Web Audio API

### Tonal Framework
- **Base Frequency:** A220 (220Hz) - low A
- **Scale:** Pentatonic minor for cohesion (A, C, D, E, G)
- **Span:** 2-3 octaves (110Hz - 880Hz)
- **Synthesis:** Simple waveforms (sine, triangle, square, sawtooth)

### Sound Map

**Player Actions:**
- Move: Subtle low pulse (110Hz triangle wave, 50ms)
- Shoot: Quick ping (440Hz sine, 80ms, slight pitch bend down)
- Hit Enemy: Satisfying "pop" (330Hz ‚Üí 220Hz, 100ms)
- Take Damage: Harsh drop (440Hz ‚Üí 110Hz sawtooth, 200ms)
- Death: Descending arpeggio (440‚Üí330‚Üí220‚Üí110Hz, 500ms total)

**Enemies:**
- Brick Spawn: Soft "bleep" (330Hz sine, 60ms)
- Brick Shoot: Sharp "pew" (550Hz triangle, 70ms)
- Bomber Approach: Low rumble (110Hz square with LFO, continuous)
- Bomber Bomb Drop: Deep "thunk" (165Hz, 120ms)

**Octo Alien:**
- Idle Ambience: Slow LFO on 220Hz (ominous presence)
- Phase Change (50% health): Dramatic tone shift (chord: 220Hz + 277Hz + 330Hz)
- Movement: Swelling pulse synchronized with position
- Drop Bomb: Threatening drop (550Hz ‚Üí 165Hz, 200ms)
- Death: Complex descending cascade (500ms, multiple frequencies)

**Charge Bomb:**
- Charging: Rising pitch (110Hz ‚Üí 440Hz over charge time, sine wave)
- Launch: Sharp "fwoosh" (880Hz spike, quick decay)
- Detonate: Layered explosion (80Hz bass + 440Hz treble, 300ms)
- Blast Active: Subtle shimmer (high frequency noise, quiet)

**UI/System:**
- Game Start: Ascending arpeggio (energizing)
- Level Complete: Victory fanfare (major scale sequence)
- Game Over: Melancholy descent

**Mix Guidelines:**
- Player sounds: Slightly louder, centered
- Enemy sounds: Balanced, slight stereo positioning based on screen position
- Bass frequencies (below 150Hz): Use sparingly for impact
- All sounds: Quick attack, short sustain, minimal release (arcade-style)

Every visual and audio element should reinforce the retro aesthetic while providing clear, satisfying feedback. The mobile-first approach means large touch targets, clear visual hierarchy, and gameplay that works in portrait orientation with one-handed controls when possible.

**Design Mantra:** *Retro aesthetics, modern feel. Simple to learn, satisfying to master.*
